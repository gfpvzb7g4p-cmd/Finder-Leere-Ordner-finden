# Finder-Leere-Ordner-finden

Dokumentation zur App „Find Empty Folders“ für das Auffinden leerer Ordner im Finder unter macOS.

## Überblick

Diese Anleitung beschreibt die Nutzung der macOS-App „Find Empty Folders“, mit der sich leere Ordner zuverlässig finden und entfernen lassen.  
Der Fokus liegt auf der Anwendung im Finder und in iCloud-Drive-Verzeichnissen.

## Vorgehensweise

1. App herunterladen und starten  
2. Einen Ordner aus dem Finder per Drag-and-Drop in das App-Fenster ziehen  
3. Die Analyse abwarten  
4. Gefundene leere Ordner prüfen und löschen

## Hinweis

In den App-Einstellungen sollte die Option  
„Don’t look inside invisible folders“ aktiviert bleiben, damit Systemordner nicht berücksichtigt werden.
